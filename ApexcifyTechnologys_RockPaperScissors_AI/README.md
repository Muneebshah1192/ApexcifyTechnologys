# 📌 Project Overview

This repository contains a **highly advanced, web-based implementation of the classic Rock–Paper–Scissors game**, developed as part of the **Artificial Intelligence Internship Program at Apexcify Technologys**.

The system simulates a **competitive human-like opponent** by analyzing previous game data and dynamically adjusting its decisions.

# 🧠 Core AI Architecture

The intelligence engine abandons simple randomness and instead uses a **layered human-like reaction system**.

It selects the best strategy from the following models:

---

## 1️⃣ N-Gram Pattern Recognition (Sequential Prediction)

The AI builds a historical dataset of player move sequences using:

* **Bigrams (2-move patterns)**
* **Trigrams (3-move patterns)**

Example:

Rock → Paper → Scissors

If this pattern repeats, the AI predicts the next move and executes the **optimal counter move**.

---

## 2️⃣ Behavioral Psychology Model (Win-Stay / Lose-Shift)

Human players often behave predictably after winning or losing.

The AI tracks the **previous round outcome**.

Typical human behavior:

| Outcome | Expected Behavior        |
| ------- | ------------------------ |
| Win     | Player repeats same move |
| Lose    | Player switches strategy |
| Draw    | Player changes pattern   |

The AI anticipates these reactions and counters accordingly.

---

## 3️⃣ Meta-Game Analysis

When a **draw occurs**, players often try to outsmart the AI by predicting a change.

The AI anticipates this psychological pivot and **breaks the draw loop with a strong counter move**.

---

## 4️⃣ Stochastic Fallback System

If:

* Player behavior is unpredictable
* Insufficient data exists

The AI switches to **cryptographic randomness** to prevent pattern exploitation.
